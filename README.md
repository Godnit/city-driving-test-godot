# City Driving Test

لعبة قيادة ثلاثية الأبعاد أصلية وخفيفة للأندرويد، مستوحاة فقط من فكرة اختبارات القيادة داخل المدينة. المشروع **Godot 4.6.3 حقيقي** ولا يستخدم HTML أو WebView، ولا يحتوي على أصول أو واجهات من Dr. Driving.

## النسخة الحالية 0.1

- سيارة سيدان ثلاثية الأبعاد أصلية مولدة داخل المشروع.
- فيزياء VehicleBody3D بأربع عجلات وتعليق واحتكاك.
- شارع مدينة طويل، مبانٍ، أرصفة، علامات مسار وحواجز.
- تحكم لمس: يمين، يسار، بنزين، فرامل، D/R.
- كاميرا خلف السيارة وكاميرا داخلية.
- مهمة قيادة زمنية بأربع بوابات واتجاه ومسافة.
- يعمل دون إنترنت ولا يطلب صلاحية الإنترنت.
- إعدادات Android تستهدف إبقاء APK تحت 100MB.

## التشغيل

افتح المجلد في Godot 4.6.3 ثم شغّل `main.tscn`.

تحكم الكمبيوتر للاختبار:

- W أو السهم الأعلى: بنزين
- S أو السهم الأسفل: فرامل
- A/D أو الأسهم: توجيه
- C: تغيير الكاميرا

## بناء APK عبر GitHub Actions

بعد وضع المشروع في مستودع GitHub، افتح تبويب Actions وشغّل `Build Android APK`. الناتج يظهر في Artifacts باسم `CityDrivingTest-Android-debug`.

## قيود النسخة الأولى

هذه بداية قابلة للتشغيل وليست لعبة مكتملة بمستوى لعبة تجارية قديمة. المراحل التالية ستضيف: قائمة سيارات، مرآب، إشارات مرور، سيارات مرور، اصطدامات وخصم نقاط، مهمات التفاف واصطفاف، أصوات أصلية، وتحسين شكل المدينة.
