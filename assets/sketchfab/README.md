## Required attribution

The final game credits screen and repository license file must include the model title, creator, Sketchfab source, and Creative Commons license. Sketchfab download URLs are temporary and require an authenticated Sketchfab account, so the source archives cannot be fetched anonymously by GitHub Actions.
